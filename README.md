# frontend

### 1. Project setup
```
npm install
若是出現Cannot find module ‘body-parser‘ 
npm install --save body-parser
```

### 2. Compiles and hot-reloads for development
```
npm run serve
```

### 若是出現Module not found: Error: Can't resolve 'vue-router' in
```
npm install --save vue-router@3
再 npm run serve
```
